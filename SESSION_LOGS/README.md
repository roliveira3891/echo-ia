# Echo IA - Session Logs History

Este diretório contém o histórico de todas as sessões de desenvolvimento do projeto Echo IA.

## Como usar:
- Cada arquivo representa um dia de desenvolvimento
- Nomeado como: `SESSION_[DATA].md` (ex: `SESSION_2025-12-03.md`)
- Contém tudo que foi feito naquele dia
- Útil para referência futura e histórico do projeto

## Como adicionar novos logs:

Quando terminar o dia, peça:
> "Salva o histórico de hoje"

E eu vou:
1. Criar um novo arquivo `SESSION_[DATA].md`
2. Documentar tudo que foi feito
3. Atualizar este README
